Práctica de aplicaciones Web

La práctica consiste en la implementación de un buscador de noticias a través de la API pública de currentsapi.

Los archivos necesarios para realizar la práctica están en el archivo app.zip. El archivo que se utilizará para probar la práctica está en app/index.html.

Funcionalidad

Al abrir la pagina index.html se debe ver en la sección "Top noticias" las primeras 6 noticias

Top noticias: https://api.currentsapi.services/v1/latest-news?apiKey=API_KEY

Encima de esta sección aparecen unos botones con todos los idiomas disponibles en la API

Languages: https://api.currentsapi.services/v1/available/languages?apiKey=API_KEY

Al pulsar intro sobre el campo de búsqueda, se debe realizar la búsqueda en la API de currentsapi usando los términos introducidos.

Busqueda: https://api.currentsapi.services/v1/search?keywords=terminos de busqueda&apiKey=API_KEY

Cuando se reciben los resultados se deben reemplazar en el listado "Top Noticias" con el mismo formato que en el ejemplo de top noticias cambiando el titulo por "Resultados".

Cuando se haga click en uno de los idiomas, se mostraran el top noticias para ese idioma con el mismo formato que en el ejemplo de top noticias cambiando el titulo por "Top noticias en {idioma}". Tanto en la barra de botones como en el idioma dentro de cada noticia

Top noticias idioma: https://api.currentsapi.services/v1/latest-news?language=es&apiKey=API_KEY

Cuando se haga un click en el titulo de la noticia se debe abrir la web de la noticia.

En caso de error accediendo a la API, debe mostrarse una alerta con un mensaje al usuario.

La página debe volver al formato inicial cuando se haga click en el botón "INICIO" en menu o se haga una recarga de la pagina, mostrando la sección "Top noticias".

API

Los datos utilizados por la aplicación deben obtenerse de la API Web de currentsapi.

Para acceder a la API, se debe registrar y utilizar el api key obtenido.

Y ponerlo como contenido <meta name="api-token"> que hay en el HTML. El código JavaScript debe coger el token de este elemento HTML al cargar la página.

Después, para obtener los datos necesarios se debe realizar una llamada al endpoint de búsqueda de la API (enviando el token como parámetro de la url "apiKey"): https://api.currentsapi.services/v1/search?keywords=terminos de busqueda&apiKey=API_KEY>

Estas llamadas deben implementarse en el código JavaScript utilizando el método Fetch.

Restricciones

Todo el código JavaScript debe ser escrito por el alumno. Se deben crear tantos archivos como sea necesario para mantener el código lo más estructurado posible.

El archivo HTML sólo puede modificarse para eliminar el código de ejemplo, añadir identificadores para poder referenciar el código fácilmente desde JS y, opcionalmente, para añadir las plantillas realizadas a partir del código de ejemplo.
